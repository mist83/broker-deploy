Editorial Theme

MULLMANIA

edited

Typography-led theme for branded surfaces that want paper, ink, cobalt, and strong role separation between display, UI, and metadata.

Usage

<script src="https://ui.mullmania.com/ui.js" data-ui-theme="editorial"></script>

Type System

  • Manrope for UI and body copy
  • Instrument Serif for display moments
  • IBM Plex Mono for labels and diagnostics
  • Role variables instead of one-off selectors

Characteristics

  • Paper and ink surfaces with one cobalt accent
  • Cardless layout bias
  • Strong brand lockups and calmer operational chrome
  • Better fit for docs, launch surfaces, and editorial UI
