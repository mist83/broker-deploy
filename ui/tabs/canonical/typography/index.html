Typography Patterns

Every text size, weight, and style the framework provides.

20 Heading Scale (h1-h6)

All headings use --weight-semibold, line-height 1.2, margin-bottom --space-md.

h1 - Page Title (36px)

h2 - Section Heading (24px)

h3 - Subsection (20px)

h4 - Group Label (18px)

h5 - Small Heading (16px)
h6 - Tiny Heading (14px)

21 Body Text & Paragraphs

Paragraphs have margin-bottom --space-md and line-height 1.6.

This is a standard paragraph with default styling. The font family is the system stack: -apple-system, BlinkMacSystemFont, Segoe UI, Roboto. Base font size is 14px.

This paragraph uses .text-secondary for less prominent content. Color is #666666.

This paragraph uses .text-muted for the least prominent content. Color is #9ca3af.

23 Code & Pre Blocks

Inline code uses --font-mono with a tertiary background. Pre blocks have a left border.

Use UI.createShell(...) to initialize the shared shell after loading.

// Pre block with left border accent
const shell = UI.createShell({
    tabsContainerId: 'tabs-container',
    contentContainerId: 'content-container',
    sitemap: sitemapData
});
await shell.init();
