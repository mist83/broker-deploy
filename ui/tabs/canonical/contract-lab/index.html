Contract Shape

Use `page` for explicit JSON structure or `preset` for a built-in specimen. Query params can override document theme and mode without changing the JSON itself.

Key Type Notes
version number Use 1.
theme string Optional document default. Query param still wins.
mode string Optional document default. Supports light and dark.
page object Use shared components, safe tags, and declarative actions.
preset string Optional shortcut to a built-in specimen.
spec64 query param Base64url encoded contract for share links and iframe previews.
