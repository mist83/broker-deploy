Button Patterns

Every button variant. All buttons lift 1px on hover and drop back on active.

28 Primary Button (default)

The default button style. Uses --color-primary background, white text.

29 Secondary Button

Tertiary background, dark text, border. For less prominent actions.

30 Danger Button

Red background. For destructive actions like delete.

31 Cancel Button

Uses --color-secondary (yellow), black text. For cancel/back actions.

32 Link Button (.btn-link)

Looks like a text link. No background, underlined, small padding.

33 Disabled Buttons

Gray background, muted text, no-pointer cursor. Works on all variants.
