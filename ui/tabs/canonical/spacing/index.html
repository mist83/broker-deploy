67 Responsive Breakpoints

Three breakpoints control layout adaptation.

Breakpoint Range Key Behaviors
Mobile <768px Grids stack to 1 col, sidebar collapses, buttons full-width, modal 95% width
Tablet 769-1024px grid-4 becomes 2 cols, sidebar narrows to 200px
Desktop >1024px Full layout, sidebar 250px, all grid columns active
