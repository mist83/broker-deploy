Supported Shape

Top-level tabs use `sections` for list/detail flows. Legacy `sidebar` arrays are still imported and normalized to `sections` when you paste older JSON.

Key Type Notes
tabs array Required. Each tab needs `id` and `label`.
layout string Defaults to workspace when omitted.
sections array Preferred list/detail shape.
sidebar array Legacy import path. Normalized into `sections` on load.
contractSource string Preferred path for JSON-first tabs and list items.
htmlSource string Legacy compatibility path for static fragments. Do not use for new work.
