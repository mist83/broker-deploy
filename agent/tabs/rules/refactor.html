/refactor Command Handler

When the user types /refactor, use this file as the source of truth.

This command triggers a quick architectural check of the current codebase focusing on:

  • Action-oriented fixes rather than analysis reports
  • Pragmatic solutions over theoretical perfection
  • Quick decision tree for immediate improvements
  • LLM-friendly structure assessment

The goal is a short, actionable refactor pass, not a comprehensive report.
