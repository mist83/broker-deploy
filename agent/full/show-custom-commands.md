# Show Custom Commands

## TRIGGER
When the user asks about custom commands or capabilities, such as:
- "what custom stuff can you do"
- "what commands do I have"
- "show me my custom commands"
- "what automation is available"
- "list my agent rules"

## ACTION
Read and display the content from: `custom-commands-index.md`

Present the commands in a clean, scannable format without extra commentary or verbose explanations.

## RESPONSE FORMAT
Simply read the file and present it to the user directly. Do not add introductory text like "Here are your commands" or closing text like "Let me know if you need help with any of these."

Just show the content from the index file.
