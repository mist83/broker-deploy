Gitter for Mac

Gitter watches repos under ~/Code, auto-commits safe mainline changes, pushes them, and keeps a menu bar status view running in the background.

Download Mac Bundle Open Repo Open Local Dashboard

What It Does

  • Watches Git repos under ~/Code.
  • Auto-commits and pushes main and master.
  • Starts itself at login with launchd.
  • Keeps a local loopback dashboard at 127.0.0.1:9477.

Install On Mac

  1. Download gitter-mac.zip.
  2. Open Install Gitter.command.
  3. Gitter starts now and again at every login.

The bundle is an unsigned local build. If Gatekeeper blocks the app the first time, right-click it once and choose Open.

Manual Install

python3 -m pip install --user -r requirements.txt
/bin/zsh install-launchd.sh
/bin/zsh install-menubar-login.sh
/bin/zsh install-web-launchd.sh

Local URLs

  • http://127.0.0.1:9477/
  • http://127.0.0.1:9481/

The web wrapper at 9481 is optional and only runs when dotnet is installed locally.
